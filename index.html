Usage

1. Unzip game_clone.zip using a tool like 7-Zip (or your OS's built-in unzip)
2. Open a terminal in the extracted folder
3. python main.py

What it does

+
Full instance copyReads the entire object tree of a place file, not just the top-level structure.
+
Script preservationKeeps Script, LocalScript, and ModuleScript source intact in the duplicate.
+
Local, offline outputOutputs a zip file containing the games full source code.

How it works

01
Point it at a place idPass the place id to the game you want to clone
02
Parse the object treeThe script walks every instance and its properties, scripts included.
03
Write the cloneA new, independent place file is saved to the output path you chose.

Built for backing up and iterating on games you already own — not for copying anyone else's work.
